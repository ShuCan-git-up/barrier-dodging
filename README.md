# barrier-dodging
躲避障碍

## 游戏说明

一个简单的网页版障碍躲避小游戏！

### 玩法
- 点击屏幕或按空格键让角色跳跃
- 躲避新冠病毒障碍
- 游戏时长3分钟，每2秒出现一个障碍
- 得分 = (已躲过障碍数 / 总障碍数) × 100
- 碰到障碍游戏结束

### 功能
- ✅ 开始游戏
- ✅ 暂停/继续
- ✅ 重新开始
- ✅ 实时得分显示
- ✅ 碰撞检测
- ✅ 游戏结束提示
- ✅ 暖心问候语（躲过随机数量障碍后显示）

### 如何开始游戏

#### 方法一：在线游玩（推荐）
1. 访问 GitHub Pages 地址：`https://shucan-git-up.github.io/barrier-dodging/`
2. 点击页面上的绿色"开始游戏"按钮
3. 使用空格键或点击屏幕让角色跳跃，躲避障碍

#### 方法二：本地运行
1. 下载或克隆本仓库
2. 直接用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮开始玩

#### 游戏操作
- **跳跃**：按空格键或点击游戏画面
- **暂停**：点击"暂停"按钮
- **重新开始**：点击"重新开始"按钮
- **目标**：躲避所有新冠病毒障碍，坚持3分钟获得满分！

#### 特色功能
- **暖心问候语**：在游戏过程中，每当你成功躲过随机数量（5-15个）的障碍后，会出现一条随机的暖心问候语，为你加油打气！问候语库包含30条不同的鼓励话语。
